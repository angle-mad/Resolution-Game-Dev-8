# Resolution Game Dev Week 8 Project

## How to Play
Use arrow keys to move the snake. Press the restart button to restart the game and your score. 

## Description
You play as a snake trying your best to eat as much fruit as possible. When eating fruit you change color and start to get longer which makes it harder to eat lots of fruit. Your goal as a snake is to eat as much fruit as you can without hitting the walls of the board or hitting yourself.
